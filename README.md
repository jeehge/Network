```Alamofire is an HTTP networking library written in Swift.```

<br>

Alamofire는 iOS, macOS를 위한 스위프트 기반 HTTP 네트워킹 라이브러리이다.

Apple의 Foundation networking 기반으로 인터페이스를 제공하여 일반적인 네트워킹 작업을 쉽게 할 수 있다.
